# 👋 Greeting App

A simple and responsive Greeting App built with **HTML**, **CSS**, and **JavaScript**.

Users can enter their name and receive a personalized greeting based on the current time of day.

## ⚙️ How It Works

1. Enter your name.
2. Click the **Show Greeting** button.
3. The app checks the current system time.
4. A greeting message is displayed based on the time of day.

Example:

```
Welcome Negar
Good Morning ☀️
```
